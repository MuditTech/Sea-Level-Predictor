# Sea Level Predictor

This repository contains my solution for the **Sea Level Predictor** project from the freeCodeCamp *Data Analysis with Python* certification.[web:55][web:62]

The project analyzes a dataset of global average sea level change since 1880 and uses **linear regression** to predict sea level rise through the year 2050 using **Pandas**, **Matplotlib**, and `scipy.stats.linregress`.[web:54][web:59]

## Project link

FreeCodeCamp project description:  
https://www.freecodecamp.org/learn/data-analysis-with-python/data-analysis-with-python-projects/sea-level-predictor[web:54][web:59]

## Files in this repository

- `sea_level_predictor.py` – main project file with the `draw_plot()` function that creates the scatter plot and two lines of best fit.
- `main.py` – entry point to run the visualization and execute unit tests locally.[web:59][web:61]
- `test_module.py` – tests (copied from the freeCodeCamp boilerplate) to verify the solution.[web:55]
- `epa-sea-level.csv` – sea level dataset used for the analysis (same as freeCodeCamp’s version), or loaded directly via the raw GitHub URL.[web:55][web:60]
